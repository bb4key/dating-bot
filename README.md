# 💘 Dating Bot — Telegram

Полнофункциональный бот для знакомств в стиле Davinchik.

## ✨ Функции

- 📋 **Регистрация** — имя, возраст, пол, предпочтения, город, описание, фото
- 👀 **Просмотр анкет** — свайп с лайком ❤️ / дизлайком 👎 / суперлайком ⚡
- 💕 **Матчи** — взаимный лайк → уведомление обоим
- 💬 **Чат** — переписка через бот между теми, кто подошёл друг другу
- 👤 **Редактирование анкеты** — любое поле в любой момент
- 🔴 Скрыть/показать свою анкету
- 🚩 **Жалобы** — репорт на пользователей
- 🛠️ **Панель админа** — бан, разбан, статистика, жалобы

## 🚀 Деплой на Railway

### 1. Создай бота в Telegram
1. Напиши [@BotFather](https://t.me/BotFather)
2. `/newbot` → задай имя и username
3. Скопируй токен

### 2. Деплой на Railway

1. Залей код в GitHub репозиторий
2. Зайди на [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий
4. Перейди в **Variables** и добавь:

```
BOT_TOKEN=твой_токен_от_botfather
ADMIN_IDS=твой_telegram_id
```

> Свой Telegram ID узнай у @userinfobot

5. Railway автоматически определит конфиг из `railway.toml` и запустит бот

### 3. Персистентное хранилище (важно!)

Railway по умолчанию сбрасывает файловую систему. Чтобы база данных сохранялась:

1. В проекте Railway → **New** → **Volume**
2. Примонтируй к `/app/data`
3. Добавь переменную:
```
DB_PATH=/app/data/dating.db
```

## 🗂️ Структура проекта

```
dating_bot/
├── bot.py              # Точка входа
├── config.py           # Конфигурация
├── database.py         # Все операции с БД (SQLite)
├── states.py           # FSM состояния
├── keyboards.py        # Клавиатуры (Reply + Inline)
├── handlers/
│   ├── start.py        # /start, главное меню
│   ├── registration.py # Регистрация (FSM)
│   ├── profile.py      # Моя анкета + редактирование
│   ├── browse.py       # Просмотр анкет, лайки
│   ├── matches.py      # Список совпадений
│   ├── chat.py         # Чат между пользователями
│   ├── settings.py     # Настройки
│   └── admin.py        # Команды администратора
├── requirements.txt
├── railway.toml
└── .env.example
```

---

## 🛠️ Локальный запуск

```bash
# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Заполни BOT_TOKEN и ADMIN_IDS

# Запусти
python bot.py
```

## 👑 Команды администратора

| Команда | Описание |
|---------|----------|
| `/admin` | Панель администратора |
| `/ban [id]` | Заблокировать пользователя |
| `/unban [id]` | Разблокировать пользователя |
| `/stats` | Статистика (пользователи, лайки, матчи) |
| `/complaints` | Последние жалобы |
